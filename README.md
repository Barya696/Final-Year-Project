- 🔭 I’m currently working on [University Teaching Hours Management and Tracking System (UTHMTS)](https://github.com/Barya696/Final-Year-Project.git)

- Description: **UTHMTS is a full-stack web application developed to digitize and automate the management of university teaching hours at the Faculté des Sciences de l'Éducation (FASED), Université de N'Djamena. The system replaces manual paper-based processes with a centralized platform for timetable management, teaching session recording, workload classification, approval workflows, and financial tariffication. It features real-time chevauchement (scheduling conflict and duplicate session) detection, role-based dashboards for administrators and academic staff, automated CM-to-TD conversion, audit logging, and notification management. Built with Spring Boot, React with TypeScript, and PostgreSQL, UTHMTS improves administrative efficiency, accuracy, transparency, and institutional accountability in higher education operations. :contentReference[oaicite:0]{index=0}**
